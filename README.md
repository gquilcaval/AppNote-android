# APP DE NOTAS (BASICO)

Proyecto educacional para gente que quiere aprender como se hace un APP de notas en kotlin.

El proyecto fue realizado con fines educativos
### Tecnologias usadas

- Room
- SQLite
- Live data
- View Model
- View Binding
